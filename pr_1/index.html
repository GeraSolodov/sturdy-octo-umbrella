Торжество (Festen. Dogme # 1)

Драма Томаса Винтерберга, первый фильм из проекта «Догма 95».

Описание | Информация | Актеры | Постер | Подробнее об актёрах

Описание

В шикарном загородном особняке намечается торжество. На юбилей почтенного отца большого семейства съезжаются дети и родственники. В зале накрыт изумительный стол, откупорены изысканные вина. За столом нет только одной из дочерей юбиляра, покончившей с собой при таинственных обстоятельствах.
